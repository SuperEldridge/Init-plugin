# 磁盘空间与临时文件防爆指南

当运行 `run_in_background=true` 的 Bash 命令时，请遵守以下规范：
1. 警觉后台任务：使用 run_in_background=true 的 Bash 命令时，必须意识到输出会被写入临时文件
2. 及时检查：后台任务完成后，应立即检查输出文件大小，异常大的文件（>100MB）需立即排查
3. 定期清理：主动检查和清理 C:\Users\<用户名>\AppData\Local\Temp\claude\ 目录下的旧任务输出
4. 输出控制：对于可能产生大量输出的命令，考虑使用 head/tail 限制输出，或将输出重定向到 nul
5. 空间监控：在执行可能产生大文件的操作前，先检查磁盘剩余空间
